# 🔐 Encrypted Chat App (RSA + AES)

A simple end-to-end encrypted chat application using:
- ✅ RSA for secure key exchange
- ✅ Fernet (AES) for fast symmetric encryption
- ✅ Sockets + Multithreading
- ✅ Optional GUI via Tkinter
